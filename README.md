# Celnet AI Lab
> AI 技术分享与落地实践 · 代码、文章、教程 统一呈现

一个面向开发者、研究者和 AI 爱好者的轻量级知识库。这里不追求复杂，而是让你能够直接在 GitHub 上阅读、学习和复用。

- 🔍 主题覆盖模型原理、工程实践、数据与应用
- 📄 以 `README.md` 作为主入口，支持 GitHub 直接预览
- 🚀 同时保留文章与代码示例，便于持续更新和整理

## 目标
Celnet AI Lab 的目标是建立一个简单、清晰、可持续维护的 AI 知识分享仓库。你可以在这里发布：

- AI 原理与技术笔记
- 实战教程与落地方案
- 代码示例、工具脚本和小型项目

## 内容分类
本仓库采用主题分类方式组织内容，便于后续沉淀与查阅：

### 文章与教程
- `articles/`：记录 AI 文章、概念总结、案例分析、实战教程
- 适合写技术笔记、经验分享、模型解析

### 代码与示例
- `code/`：放置示例项目、脚本、实验代码、可复现案例
- 适合保存与文章配套的代码、配置与使用示例

## 如何浏览
1. 打开仓库首页 `README.md`
2. 直接点击目录中的链接查看文章与代码入口
3. 如果希望阅读某个主题，在 `articles/` 或 `code/` 中查找对应章节

> GitHub 预览无需额外构建，直接在浏览器中打开即可阅读。

## 如何贡献
欢迎你持续完善本仓库：

1. 在 `articles/` 中添加新的 AI 文章或教程
2. 在 `code/` 中补充示例代码和实验项目
3. 使用简洁的标题、明确的分类，并保持内容可复现

如果你希望后续扩展为 GitHub Pages，仓库结构已经为未来升级保留了空间。
